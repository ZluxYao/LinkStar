# LinkStar

NAT 穿透 & 端口映射工具，让内网服务无需手动配置路由器即可对外暴露。

## 功能

- **STUN 穿透** — 自动检测公网 IP 和外部端口
- **UPnP 映射** — 自动在网关上创建端口映射规则
- **TCP / UDP 双支持** — 兼容 SSH、HTTP、游戏、P2P 等各类服务
- **健康检查** — 每 25 秒心跳检测，断线自动重连
- **Web UI** — 浏览器管理设备与服务，SSE 实时状态推送

## 适用场景

- 家庭 NAS / 软路由对外提供服务
- 远程访问家中 SSH / 网页
- 无公网 IP 环境下的内网穿透

## 技术栈

Go · Gin · pion/stun · goupnp · Vue.js

## 快速开始

```bash
./linkstar          # 默认监听 :3333
# 浏览器打开 http://localhost:3333
```
